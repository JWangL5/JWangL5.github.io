地质时代划分 (Classification of geologic time periods )

地质时代(Geological time periods) 分为5个时期，其中常考的有2个大时期：
  • 中生代 (Mesozoic Era), 约2.5亿~6500万年前，生活在该时期的典型生物是恐龙(dinosaur)，这段时期分为3个阶段：三叠纪 (Triassic period)、 侏罗纪 (Jurassic period)、 白垩纪 (Cretaceous period)。
    • 恐龙灭绝 (extinction) 是在6500万年前的白垩纪晚期发生的，和小行星撞击有关。
    • 更早时期的二叠纪 (Permian period), 也有一场重大的灭绝，也被认为和小行星有关。
  • 新生代 (Cenozoic Era) , 约6 5 0 0 万年前~今。新生代中常考的有：
    • 更新世(Pleistocene Epoch) , 从1 8 0 万年前到1 1 0 0 0 年前，属于冰河世纪 (ice age), 人类也
    • 在这一 时期出现。除了人类，还有一些巨型动物(megafauna) , 包括美洲猎豹 ( cheetah) 、 乳齿象 (mastodon) 等。很多更新世的巨型动物已经灭绝。
    • 全新世(Holocene Epoch), 开始于11000年前，气候变暖，各种动植物竞相出现，沉积物(sediment)中可以反映出当时物种的多种多样。人类也是通过观察沉积物中的生物来进行不同地质时期的断代。 Because the climate changed, the types of plants and animals changed also. Holocene sediments contain remnants of more recent plants and animals, so it’s pretty easy to differentiate geologically between Holocene Epoch followed the Pleistocene Epoch
  • 人类世( Anthropocene Epoch) , 开 始 于 1 8 0 0 年 工 业 革 命 ( industrial revolution) 时 期 ， 地 球 进入一个由人类主导的时期。人类大量使用化石燃料 (fossil fuels),在很大程度上改变了地球。此外，另一种说法认为8000年前，农业行为开始变得广泛，这个时间点应该是人类世的开始。
