荧光寿命显微镜相图的Python绘制和解析（Phasor Plot for FLIM-FRET）

首先简单说明一下这个实验的做法：

  • 荧光寿命是指荧光分子在被激发后，能量从激发态回落至基态所需时间的一半（$t_{1/2}$），是荧光分子的固有物理性质（在没有其他因素干扰时，通常是一个常数）

1、引包，需要引入FLIM标准格式.ptu读取模块、计算模块numpy、绘图模块matplotlib和seaborn

def phasor(fpath):
	f  = PTUreader(f'ptu3/{fpath}', print_header_data = False)
	flim_data_stack, intensity_image = f.get_flim_data_stack()

	K = flim_data_stack.shape[3]+1
	T = f.head['MeasDesc_Resolution']
	omega = 2*np.pi/(K*T)

	result = []
	t = np.linspace(0,  K*T, K-1)
	for i,x in enumerate(flim_data_stack):
		for j,y in enumerate(x):
			d = y[0]
			if sum(d)<=100:
				continue
			g = [d[i]*np.cos(t[i]*omega) for i in range(len(d))]
			s = [d[i]*np.sin(t[i]*omega) for i in range(len(d))]

			vs = integrate.trapz(s,t)
			vd = integrate.trapz(d,t)
			result.append([vg/vd, vs/vd, intensity_image[i, j]])

	result = np.array(result)
	plt.figure(figsize=(10,10))
	plt.xlim([0,1])
	plt.ylim([0,1])
